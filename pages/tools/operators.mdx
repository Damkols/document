Mint Blockchain provides documentation about how to deploy your own Mint Blockchain node.

##### Minimum hardware configuration

Minimum hardware configuration required to setup a Mint full node:

CPU: 2 core

RAM: 8 GB

Storage (last updated on May 2024): 100GB

##### Deploy Mint node

When deploying a Mint node, please follow the README.md in the repository

[mint-node](https://github.com/Mint-Blockchain/mint-node)
