##### Minimum hardware configuration

Minimum hardware configuration required to setup a Mint full node:

* CPU: 2 core
* RAM: 8 GB
* Storage (last updated on June 2024): 100GB

##### Deploy Mint node

When deploying a Mint node, please follow the README.md in the repository
