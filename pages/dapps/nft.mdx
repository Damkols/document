Mint Blockchain has integrated NFTScan as the official data infrastructure for NFTs onchain.

#### NFTScan
[NFTScan](https://www.nftscan.com) is the largest NFT data infrastructure for blockchains. 

<b>NFTScan Explorer</b> is a multi-chain NFT explorer, offering streamlined and efficient NFT data
indexing for Web3 developers and users.

Check out Mint NFTScan Explorer [here](https://mint.nftscan.com)

<b>NFTScan API</b> is a Web3 developer platform offering professional multi-chain NFT data services
to blockchain projects and the next generation of fintech companies.

Check out NFTScan developer docs [here](https://docs.nftscan.com/guides/Overview/1)
